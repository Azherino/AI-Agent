# Simple AI Agent with Ollama (Local LLM)

AI Agent sederhana menggunakan **Ollama + Python** yang berjalan sepenuhnya lokal tanpa API key.

Fungsi utama:
- Menerima pertanyaan dari user
- Melakukan reasoning sederhana (step-by-step)
- Menjawab seperti agent (bukan sekadar chatbot)
- Dapat ditambahkan memory percakapan

---

## Teknologi

- Python
- Ollama
- Model default: `llama3` (bisa diganti model lain)

---
